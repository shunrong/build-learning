# Phase 24: Rollup 学习指南

## 🎯 学习目标

1. 理解 Rollup 的设计理念
2. 掌握 Rollup 配置和插件
3. 学会打包 JavaScript 库
4. 理解 Tree Shaking 原理

## 📚 核心概念

### Rollup 的定位

```
Rollup = 专为 ES Module 设计的打包工具

优势：
- Tree Shaking 最彻底
- 输出代码最简洁
- 多格式输出（ESM/CJS/UMD）
- 最适合打包库
```

## ✅ 验收标准

- [ ] 能独立配置 Rollup
- [ ] 能使用常用插件
- [ ] 能打包出多种格式
- [ ] 能理解 Tree Shaking
