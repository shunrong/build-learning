# Demo: Rollup 库打包

## 功能

演示如何使用 Rollup 打包 JavaScript 库：
- 多格式输出（ESM/CJS/UMD）
- Tree Shaking
- 极简输出

## 快速开始

```bash
npm install
npm run build
```

## 输出格式

- `dist/my-lib.esm.js` - ES Module
- `dist/my-lib.cjs.js` - CommonJS
- `dist/my-lib.umd.js` - UMD

## 学习要点

- Rollup 配置
- 多格式输出
- 库打包最佳实践
