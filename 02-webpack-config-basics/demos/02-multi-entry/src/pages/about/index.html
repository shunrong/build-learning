🤔 什么是多入口应用（MPA）？

Multi-Page Application（MPA）是指有多个独立页面的应用，每个页面有自己的 HTML 和 JS 入口。

单页应用 (SPA)

  • ✅ 一个 HTML 文件
  • ✅ 路由在前端处理
  • ✅ 页面切换无刷新
  • ⚠️ 首屏加载慢
  • ⚠️ SEO 不友好

多页应用 (MPA)

  • ✅ 多个 HTML 文件
  • ✅ 路由由服务器处理
  • ✅ SEO 友好
  • ✅ 首屏加载快
  • ⚠️ 页面切换需刷新

💡 适用场景

  • 官网、博客、电商网站
  • 需要 SEO 的项目
  • 页面间关联度低的应用
