# Demo: Biome 统一工具链演示

## 功能

演示 Biome 的核心功能：
- Lint + Format 统一
- 极速性能
- 零配置开箱即用

## 快速开始

```bash
# 安装
npm install --save-dev @biomejs/biome

# 初始化
npx @biomejs/biome init

# 检查
npx @biomejs/biome check src/

# 自动修复
npx @biomejs/biome check src/ --apply
```
