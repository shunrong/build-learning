🔧 Babel 基础配置 Demo

📝 说明

本 Demo 演示了 Babel 如何转译 ES6+ 语法。打开浏览器控制台，查看转译后的代码效果。

提示：打开 DevTools → Sources → webpack:// → src/index.js，可以看到原始源码。
查看 dist/main.js，可以看到 Babel 转译后的代码。

1️⃣ 箭头函数

源代码：

const greeting = (name) => `Hello, ${name}!`;

2️⃣ 类（Class）

源代码：

class Person { constructor(name, age) { this.name = name; this.age = age; } introduce() { return `我叫 ${this.name}，今年 ${this.age} 岁`; } }

3️⃣ 解构赋值

源代码：

const { name, age } = person; const [first, second, ...rest] = numbers;

4️⃣ 展开运算符

源代码：

const merged = { ...obj1, ...obj2 }; const combined = [...arr1, ...arr2];

5️⃣ 可选链和空值合并

源代码：

const userName = user?.profile?.name; const count = data?.count ?? 0;

6️⃣ Async/Await

源代码：

async function fetchData() { const data = await fetch('/api/data'); return data.json(); }

🔍 查看转译结果

方式1：查看控制台输出，已自动打印转译后的代码
方式2：打开 DevTools → Sources → dist/main.js，搜索对应函数名
方式3：运行 npm run build，查看 dist/main.js 文件
注意：Babel 只转换语法，不提供 API Polyfill。
如 Promise、Array.includes 等需要 Polyfill（在 Demo 2 中讲解）。
