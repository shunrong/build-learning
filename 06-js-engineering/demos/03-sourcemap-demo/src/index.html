注意：点击 "触发错误" 和 "异步错误" 会故意抛出错误，用于演示 Source Map 如何帮助定位问题。

🔍 如何使用

  1. 打开 DevTools（F12）
  2. 进入 Sources 面板
  3. 查看 webpack:// 目录下的源代码
  4. 点击按钮触发错误，观察错误堆栈
  5. 对比有无 Source Map 的调试体验
