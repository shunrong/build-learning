📝 配置要点

1. 多入口配置

entry: {
  index: './src/pages/index/index.js',
  about: './src/pages/about/index.js',
  // ... 更多页面
}

2. 多 HTML 插件

new HtmlWebpackPlugin({
  template: './src/pages/about/index.html',
  filename: 'about.html',
  chunks: ['about']  // 只引入 about 的 JS
})

3. 历史路由支持

historyApiFallback: {
  rewrites: [
    { from: /^\/about/, to: '/about.html' }
  ]
}
