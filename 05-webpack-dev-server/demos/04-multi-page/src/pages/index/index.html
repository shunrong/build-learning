多页面应用 (MPA) Dev Server 配置演示

🎯 什么是多页面应用？

多页面应用（MPA）是指有多个独立的 HTML 页面，每个页面有自己的入口文件。与单页面应用（SPA）不同，MPA 的页面切换会重新加载。

MPA 特点

  • ✅ 多个独立页面
  • ✅ 每个页面独立的资源
  • ✅ SEO 友好
  • ✅ 首屏加载快
  • ⚠️ 页面切换有白屏

SPA 特点

  • ✅ 单个 HTML 页面
  • ✅ 前端路由控制
  • ✅ 流畅的切换
  • ⚠️ 首屏加载慢
  • ⚠️ SEO 需要特殊处理

📦 当前配置

entry: {
  index: './src/pages/index/index.js',
  about: './src/pages/about/index.js',
  contact: './src/pages/contact/index.js',
  dashboard: './src/pages/dashboard/index.js'
}

每个页面都有独立的入口文件和 HTML 模板。
