# Demo 05: 自定义 Babel 插件集合

编写一系列实用的自定义 Babel 插件。

## 运行

\`\`\`bash
npm install
npm start                # 查看所有插件列表
npm run demo:console     # 测试移除 console 插件
npm run demo:track       # 测试自动埋点插件
npm run demo:import      # 测试按需加载插件
\`\`\`

## 插件列表

1. **remove-console** - 移除 console 语句
2. **auto-track** - 自动埋点
3. **import-on-demand** - 按需加载
